# log-grid

Experimental React/TypeScript UI component for visualizing driver hours-of-service (HOS) timelines.

## Overview

This project explores rendering log grids with:

- Work vs Rest segmentation
- Time-scaled bar visualization
- Compact vs expanded views
- Future support for multi-day navigation and regulatory overlays

The goal is to prototype a clean, high-density visualization component that could integrate into larger HOS auditing systems.

## Tech Stack

- React
- TypeScript
- Vite

## Getting Started

Install dependencies:

```bash
npm install
```

Run Dev server:

```bash
npm run dev
```

## Notes

This is a personal prototype project. Expect rapid iteration and incomplete features.
